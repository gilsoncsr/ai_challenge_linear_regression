# üìà Linear Regression Applied to Irrigation Data

This project applies **supervised machine learning** techniques using **simple linear regression** to model the relationship between **irrigation hours** and the **irrigated area per angle**. The analysis includes data loading, exploratory analysis, model construction, evaluation, and predictions.

---

## üìÇ Project Structure

- `dados_irrigacao.csv`: Dataset with 39 observations.
- `main.py` or `notebook.ipynb`: Main code with analysis and modeling.
- `README.md`: Project documentation.

- `nbformat`

---

## ‚úÖ Completed Steps

1. **Data Loading and Visualization**

   - Read the CSV file and check the first few rows.

2. **Exploratory Data Analysis (EDA)**

   - Descriptive statistics.
   - Scatter plots using `seaborn` and `plotly`.
   - Correlation analysis with heatmap.

3. **Linear Regression Model Construction**

   - Split data into training and testing sets.
   - Train the model using `scikit-learn`.
   - Print the regression line equation.

4. **Model Evaluation**

   - Calculate metrics: **MSE** and **MAE**.
   - Visualize actual vs predicted values.

5. **Residual Analysis**

   - Analyze residual distribution.
   - Normality test using **Shapiro-Wilk**.

6. **Predictions**

   - Example prediction for 15 hours of irrigation.

---

## üß† Trained Model

- **Model type:** Simple Linear Regression
- **Independent variable (X):** Irrigation Hours
- **Dependent variable (Y):** Irrigated Area per Angle
- **Regression line equation:** `Y = aX + b` (coefficients are displayed in the script)
